ChatGPT, and models like it, are really just a web browser talking to a language model.

Instead of typing into the web interface, you can interact with it programmatically.

Because every interaction with ChatGPT is idempotentAfter each prompt and response, it forgets everything. You end up back in the state you were in before you interacted with it., you have to send it all previous chats every time or it will not “remember” what was said.

Here’s what a prompt looks like as an API call

      const fs = require("fs");
      const OpenAI = require("openai");
      
      const openai = new OpenAI({
        apiKey: process.env.OPENAI_API_KEY,
      });
      
      // Read the contents of the HTML file
      const htmlContent = fs.readFileSync("index.html", "utf8");
      
      openai.chat.completions.create({
        model: "gpt-4",
        messages: [
          {
            role: "system",
            content: "You will be provided with an HTML file with English content,
                      and your task is to translate only the content into Spanish and return valid HTML.",
          },
          {
            role: "user",
            content: htmlContent,
          },
        ],
        temperature: 0.7,
        max_tokens: 8192,
        top_p: 1,
      }).then(response => {
        const messageContent = response.choices[0].message.content;
        // Save the translated content to index_es.html
        fs.writeFileSync("index_es.html", messageContent, "utf8");
        
        console.log("Translation saved as index_es.html");
      }).catch(error => {
        console.error(error);
      });

The building blocks (operations)

Translation

Pass strings from one language (human or computer) and the model returns the text translated to whatever language you request.

Extraction

Take a really long text and pull out specific information from it (names, concepts, insights)

Summarization

Process human-readable unstructured data (papers, essays, articles, books) and make it computer/program-readable

Generation

Write responses, draw charts, enumerate possibilities, suggest further transformations to text or code.

Classification

Group things by any criteria you specify or let the model decide what an appropriate set is based on the information you send it.

Agents

You can string the building blocks together. The model can call itself with information it just produced.

The model can break a big task down into smaller tasks and attempt to perform them.
